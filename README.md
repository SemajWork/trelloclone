# Trello Clone

A basic Trello-style application for personal task and board management. It includes user account creation and login, the ability to create boards, add tasks to boards, and comment on tasks. The application is intended for single-user use only.

---

## Features

- User registration and login.
- Create and delete boards.
- Add, edit, and delete tasks within boards.
- Add comments to tasks.
- Basic user interface with persistent data storage.
